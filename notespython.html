Notes on Python

Variables
The primary way to create a variable in Python is to assign it a value using the assignment operator. There is no need to declare its type. Python will infer a variable's type from the assigned object. 
name = "Max" 
age = 23
If you want to ensure that the variable is of a certain type, this can be done with casting. 
seat = str(10) 
age = int(23)
weight = float(70)
To get the data type of a variable you can use the type() function. 
print(type(age)) #Output: <class 'int'>
print(type(weight)) #Output: <class 'float'>
